== Description ==
Change the backgrounds colors globally or for a specific page.
Who are we? This plugin was developed by <a href="https://allcatering.ca/" rel="noopener noreferrer" target="_blank">Catering Canada</a>, as we needed to change the background color for some of our website pages. After developing the plugin we thought to share it with the community.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' page in WordPress.
3. No extra steps are required once you activate the plugins
